# Folder Structure 
  `app` : contain nest js appliaction 
  `front` : Contain front end appliaction 

# Front end run 
    `cd front`
    `yarn`
    `yarn dev`

# Backend running the application
    `cd app`
    `yarn`
    `yarn run start`

## PreRequities 
    1. latest `nodejs`
    2. Mongodb community
